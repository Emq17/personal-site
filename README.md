# Personal Portfolio

A modern personal website and portfolio built with **React**, **TypeScript**, and **Vite**, focused on presenting my work, projects, dashboards, and hobbies in one place.

## Pages
- **Work** (`/`) — intro + skills + dashboards + experience + contact CTA
- **Home** (`/home`) — overview, about me, values
- **More Projects** (`/more-projects`) — software, financial markets, analytics
- **Hobbies** (`/hobbies`) — chess, billiards, ping pong, golf, martial arts, dancing, flipping, working out, videography, music, travel
- **Contact** (`/contact`) — contact page

## Tech Stack
- React
- TypeScript
- Vite
- React Router
- Tailwind CSS + DaisyUI

## Project Structure

### Routes (`src/pages/`)
- `Work.tsx`
- `Home.tsx`
- `MoreProjects.tsx`
- `Hobbies.tsx`
- `ContactPage.tsx`

### Shared Components
- `src/components/shared/` — layout/navigation components (e.g., `Header.tsx`)

### Work Page Components
- `src/components/work-page/` — Work page sections/components (Intro/Hero, Skills, Dashboards, Experience, Contact CTA, etc.)
- `src/components/work-page/content/` — Work page content/data (experience + skills data)
